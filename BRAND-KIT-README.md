## KryptaFlow — Brand Kit (Ready to Upload)

Ye folder aapke socials ke liye ready assets hai.

### Files
- `apps/web/public/brand/avatar.svg` — **Profile / DP (square)**
- `apps/web/public/brand/banner.svg` — **Banner (16:9)**

### Recommended PNG export sizes (most platforms SVG accept nahi karte)
- **DP / Profile**
  - 1024×1024 (best master)
  - 512×512 (safe)
  - 400×400 (minimum)
- **Banners**
  - YouTube channel art: 2560×1440
  - X (Twitter) header: 1500×500
  - Discord server banner: 960×540 (or 1920×1080)

### SVG → PNG convert (easy)
1) `apps/web/public/brand/avatar.svg` open karo browser me
2) Right click → Save as (SVG)
3) Convert to PNG using any SVG→PNG converter (example: search “svg to png”)
4) Export upar wali sizes me
